Project 1

TogakAI drawing comparison and change detection.

Overview

This assignment appears centered on identifying revisions across drawing sheets by visually highlighting deleted, added, and altered areas. The uploaded pages show plan-based markups layered over several building sheets, including first-floor, second-floor, floor-plan, reflected ceiling plan, and foundation-plan examples. The marked regions demonstrate how AI-supported comparison can help isolate scope changes that might otherwise be missed in manual review.

The project is relevant to architecture, engineering, and construction because revision tracking is a recurring coordination challenge. A workflow like this can help teams quickly identify what changed between versions and better understand where those changes affect layout, systems, and construction sequencing.

Why it matters

AI can strengthen visual plan review.

The strongest value in this project is speed and clarity. When revisions are clearly isolated and color-marked, reviewers can focus on impact rather than spend most of their time searching for differences.

Project 2

Book report: summary, structure, and reflection.

Overview

This assignment summarizes A Path Through the Jungle by Steve Peters through the metaphor of a rational path being built through emotional chaos. The report explains the Human-versus-Chimp framework, psychological immunity, the role of values and standards, and the importance of accepting reality. It also translates those ideas into a simple action plan based on trigger awareness, values, acceptance, and habit-building.

The assignment demonstrates structured communication and the ability to turn a long-form text into an organized, high-utility summary. It also shows how AI can be used to create readable educational content with clear sections, practical lessons, and reflective framing.

Key takeaway

Understanding emotion supports better decision-making.

The report’s main message is that emotional reactivity is best managed through awareness, structure, and values rather than suppression.

Project 3

Jayhawk Schedule Reviewer custom mini-agent.

Overview

This custom GPT was designed to solve a specific construction problem: the time required to compare changes between monthly schedule updates. According to the project notes, the intended users include project managers, assistant project managers, project engineers, owner’s representatives, and third-party construction managers. The tool is meant to identify date changes, log original-duration changes over time, and generate high-level phase insights from schedule comparisons.

The uploaded example shows the GPT comparing schedule updates and identifying milestone movement, high-level phase shifts, duration compression, and key takeaways. This makes the project especially relevant as an example of how a domain-specific AI tool can accelerate review workflows while still producing information that is practical for project teams.
